浏览器自带的自动换行
  • non-CJK：会在半角空格或者连字符(-)位置自动换行，不会在长单词的中间换行
  • CJK(中日韩)：可以在任何一个CJK字体后换行
  • CJK中包含non-CJK:non-CJK部分按 non-CJK情况处理进行换行处理
  • CJK中含有标点符号时，浏览器不会让标点符号位于一个行的行首，会使标点符号和前一个字符组成一个整体进行换行。
